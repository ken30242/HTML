รูปภาพ

เหตุผลที่อยากเรียน Full Stack Web Development

ผมสนใจการพัฒนาเว็บแอปพลิเคชันมาตั้งแต่เริ่มเรียนโปรแกรมมิ่ง และอยากเป็นนักพัฒนาที่สามารถสร้างเว็บไซต์ได้ครบทั้งหน้าบ้านและหลังบ้าน การเรียน Full Stack จะช่วยให้ผมเข้าใจภาพรวมของระบบเว็บแอปพลิเคชันทั้งหมด และสามารถนำความรู้ไปพัฒนาโปรเจคส่วนตัวหรือทำงานจริงในอนาคตได้ นอกจากนี้ การเป็น Full Stack Developer ยังเป็นทักษะที่ตลาดแรงงานต้องการสูงมาก

ความคาดหวังต่อวิชานี้

ผมคาดหว่าจะได้เรียนรู้ทั้งการพัฒนา Frontend ด้วย HTML, CSS, JavaScript และการพัฒนา Backend ด้วยภาษาโปรแกรมมิ่งต่างๆ พร้อมทั้งการเชื่อมต่อฐานข้อมูล หวังว่าจะได้ทำโปรเจคจริงที่สามารถนำไปใช้งานได้ และได้เรียนรู้เกี่ยวกับ API, Authentication, และ Deployment นอกจากนี้ยังอยากได้ความรู้เกี่ยวกับ Best Practices ในการเขียนโค้ดที่สะอาดและมีประสิทธิภาพ

จุดอ่อนและแนวทางการพัฒนา

จุดอ่อนของผม:

  • ยังไม่มีประสบการณ์ในการเขียน JavaScript มากนัก โดยเฉพาะ Asynchronous Programming
  • ความเข้าใจเกี่ยวกับระบบฐานข้อมูลและ SQL ยังไม่ลึกซึ้งเพียงพอ
  • ยังไม่เคยใช้งาน Git และ GitHub ในโปรเจคจริง
  • ทักษะในการ Debug โค้ดเมื่อเจอปัญหายังไม่ดีพอ

แนวทางการพัฒนา:

  • ฝึกเขียนโค้ด JavaScript ทุกวันอย่างน้อย 1 ชั่วโมง
  • ศึกษาเพิ่มเติมจาก Online Course เช่น freeCodeCamp, Udemy, YouTube
  • ทำโปรเจคเล็กๆ เพื่อฝึกทักษะ เช่น To-do List, Calculator, Weather App
  • อ่าน Documentation และฝึกใช้งาน Git/GitHub อย่างสม่ำเสมอ
  • ถามคำถามในชั้นเรียนและหาข้อมูลเพิ่มเติมเมื่อไม่เข้าใจ
  • เข้าร่วมชุมชน Developer เพื่อแลกเปลี่ยนความรู้เเละถามรุ่นพี่ที่รู้จัก

Website ที่ได้รับแรงบันดาลใจ

Website ที่ผมชอบและได้รับแรงบันดาลใจคือ: GitHub

เหตุผลที่ชอบ: GitHub มี UI/UX ที่สะอาดและใช้งานง่าย เป็นแพลตฟอร์มที่นักพัฒนาทั่วโลกใช้ร่วมกัน มีระบบ Version Control ที่ทรงพลัง และยังเป็นแหล่งเรียนรู้จากโค้ดของคนอื่นได้อีกด้วย การออกแบบที่เรียบง่ายแต่ครบฟังก์ชันทำให้ผมอยากพัฒนาเว็บแอปพลิเคชันที่มีคุณภาพแบบนี้
